#  AI Smart Water Bottle

An ESP32-based smart water bottle that tracks water intake and improves hydration using IoT and AI.

##  Features
- Automatic water consumption tracking
- Tilt-based drinking detection
- Hydration score calculation
- AI routine generator
- Manual + auto reminder system
- OLED display interface
- Web dashboard analytics
- Battery percentage monitoring

##  Hardware Used
- ESP32
- Load Cell + HX711
- MPU6050
- SSD1306 OLED
- TP4056 + Li-Po battery

##  Website Features
- Real-time dashboard
- Graphs & analytics
- AI hydration routine
